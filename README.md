### Dependencies

> SKOFL: `skofl_19b`
> 
> ATMPD: `ATMPD_19b`
>
> ROOT: `v5.28.00h`
> 
> TMVA: `v.4.2.0`

### How to install
```
git clone https://github.com/seunghophysics/NTag.git
```
```
cd NTag; make
```

### How to run
```
./bin/main -in (input filename) -out (output filename)
```
